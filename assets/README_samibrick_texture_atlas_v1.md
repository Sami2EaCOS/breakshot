# Sami / Breakshot Texture Atlas v1

Atlas PNG transparent 1024x1024, nettoye pour la version actuelle du jeu.

Il ne contient que les sprites utilises en jeu :
- vaisseaux, tourelles et boucliers ;
- briques bleues/rouges ;
- balle et projectiles sniper ;
- icones de bonus `split`, `shield`, `rapid`.

Les sprites gardent les memes coordonnees que l'ancien atlas pour eviter de modifier les references dans `scripts/Main.gd`.

Le rendu est volontairement plus net : contours propres, peu de glow, pas de lumiere diffuse qui bave sur les bords.
