# Placeholders remplaçables

Ces PNG sont utilisés par `scripts/Main.gd` pour les briques, la balle, les canons, les projectiles et les power-ups.

Pour remplacer un asset rapidement :
1. gardez le même nom de fichier ;
2. gardez idéalement un PNG transparent ;
3. ouvrez le projet dans Godot pour laisser l'import se régénérer.

Les dimensions peuvent changer : le rendu redimensionne les textures dans les rectangles de gameplay.
